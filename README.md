# Health Prediction Application

## Overview
A Health Prediction Application developed using Python, Streamlit, SQLite, and a custom machine learning prediction model.

## Features
- Add Patient Records
- Health Risk Prediction
- Search Patient
- Update Remarks
- Delete Records
- Export Records to CSV
- Health Analytics Dashboard

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py
